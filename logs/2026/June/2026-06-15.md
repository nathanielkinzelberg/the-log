- `src/` — React frontend (Bookcase → Book → LockModal → BookViewer)
- `scripts/build-logs.js` — generates `src/data/logs.json` from the markdown files

Connected the nightly cloud routine to `the-log` on GitHub. Routine runs at 11:59pm Israel time, reads today's session from the Google Drive training log, drafts the markdown entry, and commits. Push is manual for now.
